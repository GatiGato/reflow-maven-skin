Bootswatch Darkly theme

This page is using a free Darkly theme, courtesy of Bootswatch.

To use this theme with Reflow Maven skin, use the following <theme> configuration in the site.xml site descriptor file:

<custom>
  <reflowSkin>
    ...
    <theme>bootswatch-darkly</theme>
    ...
  </reflowSkin>
</custom>
